# Customer Sales Analysis

## Project Overview
This project analyzes sales and customer churn data to identify business trends, top-performing products, regional sales performance, and customer retention patterns.

## Objectives
- Analyze sales performance.
- Identify top-selling products.
- Find high-value customers.
- Compare sales across regions.
- Study customer churn behavior.
- Generate business insights through data visualization.

## Setup and Installation

1. Clone the repository:
```bash
git clone <repository-link>
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the analysis notebook:
```bash
customer_analysis.ipynb
```

## Code Structure

### Data Loading
- Load sales and churn datasets using Pandas.

### Data Cleaning
- Check missing values.
- Remove duplicate records.

### Sales Analysis
- Total sales calculation.
- Average sales calculation.
- Top product identification.
- Regional sales analysis.
- Customer performance analysis.

### Churn Analysis
- Churn distribution analysis.
- Monthly charges comparison.
- Contract type analysis.

### Visualization
- Top Products by Sales chart.
- Sales by Region chart.
- Customer Churn Distribution chart.

## Results
The project successfully identified:
- Top-selling products.
- Best-performing regions.
- High-value customers.
- Customer churn trends.
